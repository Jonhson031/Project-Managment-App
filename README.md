# Project Manager App

A simple and interactive Project Manager application built with React and Tailwind CSS. This app allows users to create projects, manage tasks, and organize their workflow in a clean UI.

## Live Demo
https://maksym-yaremko-project-managment-app.netlify.app/

---

## Features

- Add new projects  
- Delete existing projects  
- Add tasks to individual projects  
- View project details  
- Responsive and minimal UI  

---

## Technologies Used

- React  
- Tailwind CSS v3  
- JavaScript (ES6+)  

---

## What I Learned

This project helped reinforce key React concepts and modern frontend practices, including:

- Managing state using `useState`  
- Accessing DOM elements with `useRef`  
- Handling side effects with `useEffect`  
- Passing and structuring data using props  
- Creating modals using React Portals  
- Structuring reusable components
